How to download:
  1. Click Download Mods — Google Drive opens in a new tab.
  2. Click the download icon (⬇) in the top-right corner of the page.
  3. If Google shows a "Can't scan this file for viruses" warning, click "Download anyway".
  4. The real file will then download correctly.

⚠️ If your browser saves a file ending in .adding, that is Google's warning page — not the mod file. Follow the steps above and use "Download anyway".
